使用 GPT 自动化翻译

</div>

### 运行

使用 `garron-i18n md` 命令自动生成 i18n 文件 :

```shell
$ garron-i18n md
```

## 本地运行

```bash
$ git clone https://github.com/geallenboy/garron-cli.git
$ cd garron-cli
$ pnpm install
$ cd packages/garron-i18n
```
