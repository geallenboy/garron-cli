<h1>garron i18n</h1>

使用GPT自动化翻译

### 运行

使用 `garron-i18n md` 命令自动生成i18n文件 :

```shell
$ garron-i18n md
```


## ⌨️ 本地运行

```bash
$ git clone https://github.com/geallenboy/garron-cli.git
$ cd garron-cli
$ pnpm install
$ cd packages/garron-i18n
$ pnpm dev
```
