## 📦 Installation



To install Lobe UI, run the following command:



```bash
bun add @garron/cli-ui
```

## ⌨️ 本地开发


```bash
$ git clone https://github.com/geallenboy/garron-cli.git
$ cd garron-cli
$ bun install
$ cd packages/garron-cli-ui
$ bun dev
```
