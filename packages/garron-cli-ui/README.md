<h1>Garron CLI UI</h1>

Garron CLI UI 是一个用于构建\_AIGC_CLI 应用程序的开源 UI 组件库

</div>

## 安装

To install Garron UI, run the following command:

```bash
pnpm add @garron/cli-ui
```

## 本地开发

```bash
$ git clone https://github.com/geallenboy/garron-cli.git
$ cd garron-cli
$ pnpm install
$ cd packages/garron-cli-ui
$ pnpm dev
```
